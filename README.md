# CODSOFT C++ Internship Projects

This repository contains the C++ programming tasks completed as part of the CODSOFT internship program.

## Tasks Completed

### 1. Number Guessing Game
A console-based C++ program that generates a random number and allows the user to guess it.

**Features:**
- Random number generation
- User input handling
- Feedback for high/low guesses
- Continuous attempts until correct guess

---

### 2. Simple Calculator
A basic calculator application developed in C++ to perform arithmetic operations.

**Features:**
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly console interaction

---

### 3. To-Do List Manager
A console-based task management system built using C++.

**Features:**
- Add new tasks
- View all tasks
- Mark tasks as completed
- Remove tasks
- Menu-driven interface

---
